



## Anime List object formats

### Playing Status Values
  - Playing
  - Ended
  - Not Released
  - Canceled
  - Other


### Type Value
  - TV
  - Movie
  - OVA
  - Special
  - ONA
  - 

### Age Rating
  - PG
  - PG-13
  - R
  - R+


### Score Format
  - Exactly up to two decimal places : `example : 8.00`

### Time Format
  - Time with hours
  ` example :   ['1 Hour and 45 minutes'] -->  parameters : 1 , 45  `
  - Time without hours
     `example :   ['24 Minutes']  --> parameters : 24`
